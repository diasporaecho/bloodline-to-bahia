# Assets

Drop your images here.

- **bloodline-to-bahia-logo.png** — optional transparent logo shown on the cover.
  If absent, the slot hides itself automatically.

To turn the three photo-essay "plates" into real photographs, add a background image to
`.essay__plate` in `styles.css` (one rule per city), or place an `<img>` inside each plate
in `index.html`. Keep imagery cinematic and editorial — avoid generic beach/spring-break stock.
