Editor's Note

More Than a Trip

Bloodline to Bahia is a family heritage journey to Brazil built around brotherhood, Carnival, Afro-Brazilian culture, ancestry, and celebration. This is about honoring the past, celebrating the present, and creating a memory the family can carry forward.
